# 旧入口跳转文件

本文件只用于兼容旧入口。

Codex 或 Kimi 新会话默认读取：

1. `README.md`
2. 当前客户端对应的适配说明：
   - Codex：`agents/codex.md`
   - Kimi：`agents/kimi.md`

只有在准备写回共享层，或边界不清时，才继续读取 `rules/policy.md`。

执行基线：

- 先说明关键假设；如果存在多个合理解释，先摆出来，不要直接猜。
- 默认选满足需求的最简单方案，不额外加功能、抽象或可配置项。
- 只做外科手术式改动；不要顺手清理周边，也不要未请求地删除死代码。
- 多步任务先转成可验证的完成标准，再按标准检查结果。
