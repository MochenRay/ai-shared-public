# Profile

本目录用于保存长期用户画像、协作偏好、阶段焦点、健康基线和自我探索索引。

公开版只保留结构模板，不包含任何真实个人信息。

## Suggested Files

- `user-profile.md`：稳定用户事实模板
- `collaboration-preferences.md`：协作偏好模板
- `current-focus.md`：当前阶段焦点模板
- `health/README.md`：健康相关长期记录入口
- `exploration/README.md`：自我探索材料入口

## Boundary

- 可放：用户明确确认、可长期复用的稳定事实和偏好。
- 不放：原始聊天记录、隐私细节、证件、账号、地址、密钥、医疗原始数据。
